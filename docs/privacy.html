プライバシーポリシー

最終更新日: 2025年11月3日

献立くん（以下「本アプリ」といいます）は、 ユーザーの個人情報の取扱いについて、以下のとおりプライバシーポリシー（以下「本ポリシー」といいます）を定めます。

第1条（収集する情報）

本アプリは、サービスの提供にあたり、以下の情報を収集します。

1. ユーザーが提供する情報

  • 選択された食材情報
  • お問い合わせ内容

2. 自動的に収集される情報

  • デバイス情報（機種、OS、バージョン等）
  • アプリの利用状況（アクセス日時、利用機能等）
  • IPアドレス
  • Cookie及び匿名ID

第2条（利用目的）

本アプリは、収集した情報を以下の目的で利用します。

  • 本アプリのサービス提供
  • 献立の生成と提案
  • ユーザーサポート
  • 本アプリの改善および新機能の開発
  • 利用状況の分析
  • 不正利用の防止
  • お問い合わせへの対応
  • 利用規約違反の対応

第3条（個人情報の第三者提供）

本アプリは、サービス提供のために、以下の第三者に個人情報を提供します。本アプリの利用により、これらの第三者提供に同意したものとみなします。

OpenAI（献立生成サービス）

  • 提供する情報: 選択された食材情報
  • 提供の目的: 献立の生成
  • 提供先の国: アメリカ合衆国

また、次に掲げる場合には、法令に基づき第三者に個人情報を提供することがあります。

  • 法令に基づく場合
  • 人の生命、身体または財産の保護のために必要がある場合であって、本人の同意を得ることが困難であるとき
  • 公衆衛生の向上または児童の健全な育成の推進のために特に必要がある場合であって、本人の同意を得ることが困難であるとき
  • 国の機関もしくは地方公共団体またはその委託を受けた者が法令の定める事務を遂行することに対して協力する必要がある場合であって、本人の同意を得ることにより当該事務の遂行に支障を及ぼすおそれがあるとき

第4条（外部サービスの利用）

本アプリでは、以下の外部サービスを利用しています。これらのサービスにおける情報の取り扱いについては、各サービスのプライバシーポリシーをご確認ください。

OpenAI API

献立の生成にOpenAI APIを使用しています。選択された食材情報は、献立生成の目的でOpenAIに送信されます。 OpenAIのプライバシーポリシーについては、こちらをご確認ください。

第5条（個人情報の開示）

本アプリは、本人から個人情報の開示を求められたときは、合理的な期間内にこれを開示します。 ただし、開示することにより次のいずれかに該当する場合は、その全部または一部を開示しないこともあり、 開示しない決定をした場合には、その旨を通知します。

  • 本人または第三者の生命、身体、財産その他の権利利益を害するおそれがある場合
  • 本アプリの業務の適正な実施に著しい支障を及ぼすおそれがある場合
  • その他法令に違反することとなる場合

第6条（個人情報の訂正および削除）

ユーザーは、本アプリの保有する自己の個人情報が誤った情報である場合には、 本アプリに対して個人情報の訂正、追加または削除（以下「訂正等」といいます）を請求することができます。

本アプリは、ユーザーから前項の請求を受けてその請求に応じる必要があると判断した場合には、 合理的な期間内に当該個人情報の訂正等を行うものとします。

第7条（個人情報の利用停止等）

本アプリは、本人から、個人情報が、利用目的の範囲を超えて取り扱われているという理由、 または不正の手段により取得されたものであるという理由により、その利用の停止または消去（以下「利用停止等」といいます）を求められた場合には、 必要な調査を行います。

前項の調査結果に基づき、その請求に応じる必要があると判断した場合には、当該個人情報の利用停止等を行います。

第8条（Cookie及び類似技術）

本アプリは、本アプリの利用状況の把握、サービスの改善のためにCookie及び類似技術を使用することがあります。 ユーザーは、ブラウザの設定によりCookieを無効にすることができますが、その場合、本アプリの一部機能が正常に動作しない可能性があります。

第9条（データの保存期間）

本アプリは、収集した個人情報を、利用目的を達成するために必要な期間、または法令等で定められた期間保存します。 保存期間経過後は、適切な方法で削除または匿名化します。

第10条（セキュリティ）

本アプリは、個人情報の正確性及び安全性確保のために、セキュリティ対策を講じています。 ただし、インターネット通信の性質上、セキュリティを完全に保証することはできません。

第11条（プライバシーポリシーの変更）

本アプリは、必要に応じて、本ポリシーを変更することがあります。 変更後のプライバシーポリシーは、本アプリ上に掲載したときから効力を生じるものとします。 重要な変更がある場合は、アプリ内での通知等により、ユーザーに周知します。

第12条（ご意見・ご要望）

アプリに関するご意見やご要望をお寄せください。
※個別の返信はいたしかねる場合があります。

連絡先
メールアドレス: support@your-domain.com（実際のメールアドレスに変更してください）

第13条（適用範囲）

本ポリシーは、本アプリにおいてのみ適用されます。 本アプリ内から外部サイトへのリンクがある場合、リンク先のサイトにおける個人情報の取り扱いについては、 本アプリは責任を負いかねます。

第14条（準拠法と管轄裁判所）

本ポリシーの解釈にあたっては、日本法を準拠法とします。
本ポリシーに関して紛争が生じた場合には、開発者の所在地を管轄する地方裁判所を専属的合意管轄とします。
